RGB vs CMYK

RGB (Red, Green, Blue)

Como funciona:

O modelo RGB é um sistema de cores aditivo. Isso significa que as cores são criadas pela combinação de luz vermelha, verde e azul em diferentes intensidades.

  • Quando todas as três cores estão no máximo de intensidade, o resultado é a luz branca.
  • Quando todas as três cores estão em zero intensidade, o resultado é o preto.
  • Cada cor (vermelho, verde e azul) pode variar de 0 a 255, resultando em milhões de combinações de cores possíveis.

Onde é usado:

O modelo RGB é usado em dispositivos que emitem luz, como monitores de computador, telas de celular, televisões e câmeras digitais. É o modelo de cores padrão para a web e para a maioria dos softwares de edição de imagem digital.

CMYK (Cyan, Magenta, Yellow, Key/Black)

Como funciona:

O modelo CMYK é um sistema de cores subtrativo. Isso significa que as cores são criadas pela remoção de luz branca através da mistura de tintas ciano, magenta, amarela e preta.

  • Quando todas as quatro tintas estão no máximo de intensidade, o resultado é o preto.
  • Quando não há tinta, o resultado é o branco (a cor do papel).
  • A tinta preta (Key) é adicionada para melhorar a qualidade das cores escuras e para economizar tinta colorida.

Onde é usado:

O modelo CMYK é usado em impressoras e em processos de impressão comercial. É o modelo de cores padrão para a impressão de materiais como livros, revistas, jornais e embalagens.

Principais diferenças:

  • Luz vs. Tinta: RGB usa luz para criar cores, enquanto CMYK usa tinta.
  • Aditivo vs. Subtrativo: RGB é um modelo aditivo, onde as cores são somadas para criar novas cores. CMYK é um modelo subtrativo, onde as cores são subtraídas da luz branca.
  • Tela vs. Impressão: RGB é usado em telas digitais, enquanto CMYK é usado em impressoras.
  • Gama de Cores: RGB possui uma gama de cores maior do que CMYK, o que significa que pode exibir mais cores diferentes.

Informação adicional:

É importante lembrar que as cores exibidas em um monitor (RGB) podem não corresponder exatamente às cores impressas (CMYK). Isso ocorre porque os dois modelos de cores funcionam de maneiras diferentes e possuem gamas de cores diferentes. Quando um designer cria uma imagem para impressão, ele geralmente precisa converter a imagem do modelo RGB para o modelo CMYK para garantir que as cores sejam impressas corretamente.
