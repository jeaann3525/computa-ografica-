Explicação dos Modelos de Cores RGB e CMYK

Modelo RGB (Red, Green, Blue)

O modelo RGB é utilizado para representar cores em dispositivos que emitem luz, como monitores e televisores. Ele funciona combinando luzes vermelha, verde e azul, em diferentes intensidades, para formar outras cores. Quando as intensidades de vermelho, verde e azul são máximas (255), a cor resultante é branca. Quando todas as intensidades são zero (0), a cor resultante é preta.

RGB(255, 0, 0)
RGB(0, 255, 0)
RGB(0, 0, 255)

Modelo CMYK (Cyan, Magenta, Yellow, Key/Black)

O modelo CMYK é utilizado em impressões, trabalhando com a subtração de luz. A combinação das cores ciano, magenta, amarelo e preto gera outras cores. Quando todas as cores são combinadas em 100%, o resultado é o preto. Este modelo é baseado em pigmentos, e as cores são formadas pela absorção de diferentes comprimentos de onda da luz.

CMYK(100%, 0%, 0%, 0%)
CMYK(0%, 20%, 100%, 0%)
CMYK(50%, 80%, 0%, 20%)
